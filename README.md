# Store

Build online store app with server side rendering using next technologies: NodeJs+ExpressJs, PostgreSQL(+ Sequelize- optional, but preferred), ejs for templating. 
All bellow functionality is required, but you also can add some more features.
To occur unexpected errors validation is preferred. 
Styling is not necessary.  
1) Authorization.     
 - Sign In   
 - Sign Up    
 - Log Out
2) Add item
3) List items:    
 - Sorting   
 - Filtering
4) Item detail view
